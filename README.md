# oracleDataTransfer

#### 介绍
利用 sqluldr2导出增量数据  使用sqlldr导入数据  通过merge into 合并更新数据。
![image](https://s1.ax1x.com/2020/07/15/UdGkr9.png)


#### 软件架构
主要分成三个shell脚本完成   oracle2File.sh 导出数据  file2Oracle 导入数据（insert直接入库，update数据入update库）   excSql 合并更新数据。
